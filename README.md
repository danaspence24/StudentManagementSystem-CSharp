# Student Management System (C# / WinForms)

## Overview 
A Windows Form application that manages student data in memory using a List<Student>. Users can add, update filter, and analyze student records using LINQ. 

## Tech Stack
- C# (.Net WinForms)
- LINQ
- Object-Oriented Programming 

## Features 
- Add, edit, and display student information
- LINQ filtering 
- Sorting by name or score
- Aggregation: Count, Average, min/max scores
- Element access using First(), Last(), ElementAt()

## What I Learned 
- Working with collections
- LINQ (method + query syntax)
- UI event-driven programming
